# react-native-HearthApp

Hearth — a React Native + Expo app that prevents food waste and eliminates dinner
decision fatigue. Scan a grocery receipt → pantry auto-populates → the Panic Button
generates a meal from expiring items → cook it → pantry deducts automatically.

## Tech stack

- Expo (managed) + Expo Router (file-based routing)
- TypeScript (strict)
- NativeWind v5 + Tailwind CSS v4 (CSS-first design tokens)
- Reanimated, Zustand
- Supabase (Auth, Postgres + RLS, Edge Functions)

## Getting started

```bash
npm install
npm start
```
